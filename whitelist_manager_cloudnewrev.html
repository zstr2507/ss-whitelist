⚡ Setup Firebase
Ikuti langkah ini untuk menyimpan data di cloud. Gratis selamanya.
1Buat Project Firebase
Buka console.firebase.google.com → Create a project
Beri nama (misal: ss-studio-whitelist) → disable Analytics → Create
2Aktifkan Authentication
Sidebar: Build → Authentication → Get started
Tab Sign-in method → klik Google → Enable → isi email → Save
3Buat Firestore Database
Sidebar: Build → Firestore Database → Create database
Pilih Start in production mode → region asia-southeast1 → Enable
Setelah dibuat → tab Rules → hapus semua → paste ini → Publish:
rules_version='2';service cloud.firestore{match/databases/{database}/documents{match/users/{uid}/{document=**}{allow read,write:if request.auth!=null&&request.auth.uid==uid;}}}
4Daftarkan Web App & Copy Config
Klik ikon ⚙️ → Project settings → scroll ke bawah → klik ikon </>
